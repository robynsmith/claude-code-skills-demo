# Research Assistant

Conduct thorough research on a topic by searching the vault, exploring connections, and synthesizing findings.

## Topic

$ARGUMENTS

## Process

### 1. Initial Search

- Search the entire vault for the topic and related terms
- Identify all relevant notes
- Note gaps in existing knowledge
- Check journal and learning log for past reflections on this topic

### 2. Deep Dive

- Read all relevant notes thoroughly
- Extract key insights and quotes
- Identify contradictions or tensions
- Map connections between ideas
- If web research would help, search for current information

### 3. Synthesis

- Create a summary of findings
- Highlight patterns and themes
- Note questions that remain unanswered
- Suggest areas for further research

### 4. Create Resource Document

- Save findings to appropriate location in `contexts/[context]/3_resources/`
- Follow existing formatting patterns
